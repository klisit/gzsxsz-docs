# 快速浏览

## 主页页面介绍
![主页介绍](/main-ind.png)


### 菜单列表

菜单列表包含，当前用户可操作的菜单，和可查询可编辑的数据。权限由管理员在菜单管理，角色管理页面配置

### 用户和消息操作区

用户和消息操作区包含，当前用户接收消息，用户信息查看和用户密码修改、登录、退出操作

### 已打开菜单操作区

已打开菜单操作区包含，当前打开的菜单，可快速浏览访问过的菜单，刷新当前菜单关闭当前菜单等

### 页面内容主区域

页面内容主区域，显示数据表单区域，执行数据查看，修改，创建，删除的地方，构成页面的主要组成部分
